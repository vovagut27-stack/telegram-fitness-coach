# Монетизация FitBot

## Уже работает

- **Telegram Stars** — подписка Premium на 30 дней (`PREMIUM_STARS_PRICE`, `PREMIUM_DAYS`).
- **Freemium** — лимит завершённых тренировок в неделю (`FREE_WORKOUTS_PER_WEEK`).

## Premium в приложении (после обновления)

| Фича | Free | Premium |
|------|------|---------|
| Домашние тренировки | 3/нед | Безлимит |
| Расписание | 7 дней | 14 дней |
| История результатов | 30 дней | 120 дней |
| Программа зала | — | 4-дневный сплит |
| Аналитика на главной | — | Объём, тренд, фокус |
| Личные рекорды (PR) | — | Топ по весу/повторам |
| Экспорт недели | — | Текст в буфер / шаринг |
| Таймер отдыха | Обычный | Короткий / длинный |
| Напоминания | Базовые | + приоритет в cron |

## Идеи на следующие релизы

1. **Пакеты Stars** — 7 / 30 / 90 дней со скидкой на длинный период.
2. **Разовая покупка** — «Разблокировать зал навсегда» отдельным инвойсом.
3. **Рефералка** — +3 дня Premium за друга, который оплатил (payload в `start=ref_ID`).
4. **Пробный Premium** — 3 дня один раз (`premium_trial_used` в БД).
5. **Платные планы питания / сон** — партнёрские ссылки (affiliate), без разработки ядра.
6. **Корпоратив / челлендж** — групповая лицензия через бота-админа.
7. **Кастомный сплит** — Premium+: выбор «ноги 2× в неделю» (настройка в профиле).
8. **Экспорт CSV** — для Google Sheets (Premium).
9. **Голосовые подсказки** — TTS между подходами (отдельный микроплатёж).
10. **Сезонные бейджи** — косметика в профиле за Stars.

## Ценообразование (ориентир)

- Free должен быть **полезным**, но упираться в лимит недели и отсутствие зала.
- Premium — цена **1–2 чашки кофе в месяц** в Stars (сейчас ~150⭐ — ок для RU/EU mix).
- A/B: 99⭐ первый месяц, 199⭐ продление.

## Метрики

- Конверсия free → paid после 2-й завершённой тренировки.
- Retention D7 у Premium vs Free.
- Доля оплат после показа PR / аналитики (тизер для free).
